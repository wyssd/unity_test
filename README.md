# GlowCore

## Contributing
Please refer to the [Coding Guidelines](CODING-GUIDELINES.md) for details on code style and best practices when contributing to this project.

Contributions must always be linked to a [Jira](https://ost-team-eos9sg4l.atlassian.net/jira/software/projects/GC/boards/1) issue. As such, branches should be named in the format:
```
<type>/gc-1234-description
```
Where `gc-1234` is the Jira issue key and `description` is a brief description of the work being done. `<type>` can be one of the following:
- `feature`: For new features or significant enhancements.
- `bug`: For fixing bugs or issues.
- `docs`: For documentation updates or improvements.
- `task`: For general tasks that don't fit into the above categories.

Pull Requests should reference the Jira issue in the description and provide a clear summary of the changes made.
