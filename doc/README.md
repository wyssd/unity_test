# Vorlage für Typst Dokus an der FH OST

Diese Vorlage wurde von Vanessa Alves und Marco Schnider während der Studien- und Bachelor-Arbeit im FS 2025 erstellt.

[Typst](https://typst.app/) hat eine sehr gute Dokumentation. Bitte lest euch selbst rein bevor ihr die Autoren für Fragen kontaktiert (diese werden sehr wahrscheinlich ignoriert). Nützliche Pull-Requests werden selbstverständlich angenommen.

Eine Online Version und Download der Dokumentation ist per GitLab Pages möglich und wird in der **.gitlab-ci.yml** Datei definiert.

Zugriff auf GitLab-Page: https://typst-template-ost-bae147.gitlab.io/documentation.pdf

Relevante Jobs:
- protocols_pdf_build: Download der aktuellen Protokolle
- main_pdf_build: Download der aktuellen Dokumentation
